# 🎨 Desarrollo Web: HTML5 & CSS3

¡Bienvenido al bloque de Fundamentos de Interfaz! En esta sección del cuaderno documento todo lo relacionado con la estructura semántica de la web y el diseño visual. Aquí es donde se construye todo lo que el usuario ve y toca.

---

## 📚 Secciones de Aprendizaje

A medida que avance el verano, iré completando los apuntes divididos en estos dos grandes bloques:

### 🏗️ 1. HTML5 (Estructura y Semántica)
El esqueleto de la web. Aquí guardo las buenas prácticas para estructurar contenido de forma accesible y óptima para buscadores (SEO).

- [Estructura global del documento, etiquetas esenciales y validación.](html/01-html-basico.md)
- [ ] Formateo de textos, listas avanzadas y enlaces.
- [ ] Tablas informativas y contenido multimedia (`<audio>`, `<video>`, `<img>`).
- [ ] Formularios accesibles y validación de datos nativa.

### 🎨 2. CSS3 (Estilos y Diseño)
La capa visual. El objetivo de este bloque es dominar cómo transformar un documento de texto aburrido en una interfaz moderna y adaptables a móviles.

- [x] Selectores, la cascada y el Modelo de Caja (*Box Model*).
- [ ] Flexbox y CSS Grid para layouts complejos.
- [ ] Animaciones, variables CSS (*Custom Properties*) y Responsive Design.
